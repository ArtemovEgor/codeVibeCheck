# 9 марта 2026 — Заканчиваю интерфейс чата

Решил, что чат с ИИ будет устроен так: пользователь может начать собеседование с ИИ-интервьюером в этом чате, и за правильные ответы на вопросы ИИ будет ачислять баллы, которые отображаются в углу экрана. К настоящему моменту не планирую делать историю чатов, только добавил возможность его сброса, чтобы начать новый чат.

Как ни странно, пришлось повозиться с кнопкой сброса чата. Придумал ей такой дизайн: изначально кнопка круглая со стрелкой, а при наведении, из нее появляется сбоку красный текст Reset. Получилось довольно симпатично, но для мобильных устройств пришлось делать другой флоу, ведь там нет наведения мышки, поэтмоу нет смысла опираться на этот эффект. Сделал так, чтобы на мобильных нужно было нажать на кнопку два раза: сначала пользователь нажимает на круглую кнопку -> из нее появляется надпись Reset -> пользователь нажимает на кнопку еще раз -> чат сбрасывается.

Я не знал, как в JS проверить, использует ли пользователь тач устройство или мышку, в итоге сделал так:

```
globalThis.matchMedia("(hover: hover)").matches;
```

Результатом я пока доволен. Органично в процессе работы начал читать про работу с API ИИ, стримингом сообщений и с фронта, и с бэка. Возможно, попробую взяться за эту задачу и на бэке тоже.
