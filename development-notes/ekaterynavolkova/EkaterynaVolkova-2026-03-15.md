### Что изучала

**src/database.ts** - инициализация базы данных SQLite через better-sqlite3.
База данных это просто файл database.sqlite в папке проекта.
При первом запуске файл создаётся автоматически, таблица "users" тоже создаётся если её нет через 'CREATE TABLE IF NOT EXISTS users'.

**backend/src/index.ts** - точка входа.
Здесь регистрируются роуты: /api/health, /api/auth/register, /api/auth/login
app.listen - используется для запуска сервера и настройки его на прослушивание входящих запросов на указанном порту
express.json() - встроенный middleware для считывания JSON данных отправленных к примеру в запросах

Потом не подтягивались переменные из .env. Разобралась что нужен импорт import "dotenv/config"

Поняла также что для запуска бека рядом с фронтом лучше использовать разные порты, а то они у меня конфликтовали

Потом обратила внимание при тестировании, что структура ответов с бека не совпадает с ожидаемой, это мы обсудили с командой и быстро поправили.
